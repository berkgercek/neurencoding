`neurencoding.poisson`: Poisson link-fuction fitting of spikes
=======================================================================

The `poisson` module consists primarily of a single member, `PoissonGLM`,
which takes in spiking information (clusters and times) and a design matrix,
then performs regression using the PoissonRegressor sklearn class.
